# 📝 To-Do List

Hello! This is a simple **To-Do List app** with a basic UI design, featuring **light and dark mode** and interactive **JavaScript logic**.  
You can add and delete tasks using buttons with a visual presentation.

---

## 🚀 Features
- ➕ Add task button  
- ❌ Delete task button  
- 🌙/☀️ Light & Dark mode  

---

## 🛠 Tech Stack
- **HTML**  
- **CSS**  
- **JavaScript**
